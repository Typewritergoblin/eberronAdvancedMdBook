# Appendix B: Monster Lists

This appendix provides conversions for stat blocks that have changed between the 2014 [Monster Manual](/sources/dnd/mm-2014) and this book, as well as lists of monsters organized by habitat, creature type, group, and Challenge Rating.

## Monster Conversions

Every monster in 2014 [Monster Manual](/sources/dnd/mm-2014) either appears in this book or has a CR-appropriate replacement. If you’re using material published prior to this book and are unable to locate a stat block herein, consult the Stat Block Conversions table to find the equivalent stat block you should use.
