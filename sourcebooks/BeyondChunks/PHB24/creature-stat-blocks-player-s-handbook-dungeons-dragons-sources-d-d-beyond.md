# Appendix B: Creature Stat Blocks

This appendix provides stat blocks for creatures mentioned elsewhere in the book, particularly in the class, equipment, and spell chapters. See the [rules glossary](/sources/dnd/phb-2024/rules-glossary#StatBlock) for how to read a [stat block](/sources/dnd/free-rules/rules-glossary#StatBlock "//www.dndbeyond.com/rules-glossary/111-tooltip?disable-webm=1&disable-webm=1"), and see the [Monster Manual](/sources/dnd/mm-2024) for even more creatures.

The following stat blocks are presented in alphabetical order. When the Dungeon Master uses a stat block, the DM may change details in it.
